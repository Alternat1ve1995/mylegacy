Usage:

1) Copy "props.bat" and "props.py" to root directory from wich you want to start search.
2) Double click on "cherrypick.bat".
3) Script will create file "build.properties" in same directory and write every property into it.

P.S. Make shure your Python 2.7 is in PATH.
P.P.S. This is not fully debugged, so it may place some crap into file also :)
